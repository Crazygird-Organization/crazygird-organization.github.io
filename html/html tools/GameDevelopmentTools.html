Game Development Tools


To make games I would recommend using the game engine Unity. Unity is a game engine that makes the process of creating games easier.
Unity uses the coding language C#.

Here is a list of the things that you may need to download.

  • Unity Hub
  • Visual Studio. Download while downloading editor.
  • .Net. Download Here.

How to install Unity Editor.

Once you have installed the Unity Hub go to the installs tab. In the installs tab click install editor at the top right.
Click install on the version that is in LTS. Select Visual Studio unless you already have it installed. Then go and select all the modules that you want.
You can download the other modules later if you want to. Click install and let the editor install.
